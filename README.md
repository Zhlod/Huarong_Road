Game文件夹为游戏文件
AI_Competition文件夹为AI大比拼文件

##Game部分使用
配置运行环境
进入Game目录 在命令行下输入
```
pip install -r requirements.txt
```
安装完后再命令行下输入
```
python game.py
```
开始游戏

##AI大比拼部分使用
配置运行环境
进入AI_Competition目录 在命令行下输入
```
pip install -r requirements.txt
```
AI_Comp.py为大比拼主要文件, 采用的是注释函数, 使用时将函数注释清除, 不使用时注释掉
